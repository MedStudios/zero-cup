## 网站结构

| 文件或目录           | 内容                          |
| ------------------- | ----------------------------- |
| `index.html`        | 主页                          |
| `history.html`      | AI编年史                      |
| `about.html`        | WST简介                       |
| `404.html`          | 404返回页面                   |
| `img/`              | 网页图片目录                  |
| `video/`            | 网页视频目录                  |
| `js/`               | 网页本地JavaScript脚本目录     |
| `css/`              | 网页本地CSS样式表目录          |
| `css/main.css`      | 所有网页通用的CSS样式表        |
| `css/normalize.css` | 统一网页在不同的浏览器上的外观  |
